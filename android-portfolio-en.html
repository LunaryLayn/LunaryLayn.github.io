Android Portfolio

Blockfolio

  • Category: Personal Project
  • GitHub Repository URL: https://github.com/LunaryLayn/Blockfolio

Android application developed in Kotlin as a personal project for managing a cryptocurrency portfolio. The application features functionalities such as user authentication with Firebase, data retrieval from APIs using Retrofit from CoinGecko, data storage in a local database with Room, responsive design with Compose, notifications, and more.

List of technologies used:

  • - Developed in Kotlin
  • - Notifications
  • - Coroutines
  • - Jetpack Compose
  • - Compose Navigation
  • - Gson
  • - Encrypted SharedPreferences
  • - RoomDB
  • - Retrofit
  • - Firebase
  • - Hilt (Dependency Injection)
  • - Coil
