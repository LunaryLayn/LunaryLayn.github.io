Blockfolio

  • Categoria: Proyecto personal
  • URL Repositorio de GitHub: https://github.com/LunaryLayn/Blockfolio

Aplicación Android desarrollada en Kotlin como proyecto personal para la gestión de una cartera de criptomonedas. La aplicación cuenta con funcionalidades como autenticacion de usuarios con Firebase, obtencion de datos de APIs con Retrofit de CoinGecko, almacenamiento de datos en una base de datos local con Room, diseño responsivo con Compose, notificaciones, y mas.

Lista de tecnologias utilizadas:

  • - Desarrollada en Kotlin
  • - Notificaciones
  • - Coroutines
  • - Jetpack Compose
  • - Compose Navigation
  • - Gson
  • - Patron MVVM
  • - Encrypted SharedPreferences
  • - RoomDB
  • - Retrofit
  • - Firebase
  • - Hilt (DI)
  • - Coil
  • - LiveData y Flows
